About the Timetable Generator

The Timetable Generator is a lightweight, intelligent web application designed to simplify academic scheduling. It automatically generates balanced class timetables based on course data, lecturer availability, and departmental configurations.

This platform allows administrators to:

  • Select departments and view course lists instantly.
  • Automatically generate fair and balanced timetables.
  • Manage lecturers, rooms, and departmental configurations from one dashboard.

The system ensures no overlapping classes and a fair distribution of lecture hours per week.
